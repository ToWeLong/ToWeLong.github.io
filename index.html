解决安装Java11不生成jre的问题
2019-10-28 1 min read # java

之前也碰到过安装JDK不会自动生成JRE的问题，但是一直没去解决，因为这个问题出现在同学的电脑上，我自己找的那个JDK是会自动生成JRE的，然后最后也是拿我这个安装包解决的。

首先进入JDK安装的根目录下，然后执行以下命令：

bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre

然后，在环境变量中配置JDK以及JRE即可。

SpringBoot解决跨域问题
2019-10-24 3 min read # SpringBoot

什么是跨域？

定义：浏览器从一个域名的网页取请求另一个域名下的东西。通俗点说，浏览器直接从A域访问B域中的资源是不被允许的，如果想要访问，就需要进行一步操作，这操作就叫“跨域”。例如，你从百度的页面，点击一个按钮，请求了新浪的一个接口，这就进行了跨域。不单单只有域名不同就是跨域，域名、端口、协议其一不同就是不同的域，请求资源需要跨域。
